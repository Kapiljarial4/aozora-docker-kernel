# Aozora Kernel with Docker Support

This repository builds the Aozora kernel with Docker/container support enabled.

## Features
- Based on Aozora Kernel 4.9.337-V7
- Docker support (namespaces, cgroups, overlay filesystem)
- Built for Poco F1 (beryllium)
- Android 13 compatible

## Build
Push to main branch or trigger workflow manually to build.

## Flash
Download the `Aozora-Docker-Kernel-*.zip` from Actions artifacts and flash via TWRP/recovery.
